# WhatTimeIsIt.io - Bug Reports & Feature Requests

Welcome! This repository is for reporting bugs and suggesting features for [whattimeisit.io](https://whattimeisit.io).

> **Note:** This is an issue-tracking repository only.

## üêõ Found a Bug?

Please [open an issue](../../issues/new) and include:

- **Description:** What happened vs. what you expected
- **Location:** Which page/city (include the URL)
- **Browser:** Chrome, Safari, Firefox, etc.
- **Screenshot:** If possible, attach a screenshot
- **Steps to reproduce:** How can we recreate the issue?

### Example Bug Report

```
Title: Nepal UTC offset shows +5.5 instead of +5:45

The displayed UTC offset for Nepal shows "+5.5 hrs" but it should show "+5:45 hrs"
since Nepal uses UTC+5:45. The actual time shown appears correct, but the offset
label is misleading.

Page: https://whattimeisit.io/asia/nepal.html
Browser: Chrome 120 on macOS
```
